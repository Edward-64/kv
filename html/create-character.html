Введите псевдоним. Обязательное поле. Псевдоним нужен для активации персонажа, поэтому его нельзя сообщать другим людям. Придумайте такой псевдоним, который Вы хорошо запомните: если забудете пароль, по примерному псевдониму можно будет восстановить доступ к персонажу.
Введите пароль. Очевидно, обязательное поле.
По желанию, укажите свою электронную почту. Если Вы забудете пароль, с помощью почты можно будет автоматически восстановить доступ к персонажу.
